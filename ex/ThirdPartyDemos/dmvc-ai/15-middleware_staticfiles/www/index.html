mORMot2 Static Files Middleware Sample

Current Path: /static (www folder)
Server: mORMot2 TRestHttpServer with custom static file serving
Features: Default file serving, MIME type detection, cache headers

Available Static Paths

Path 1: /static → www/ Path 2: /static2 → www2/ Path 3: /static3 → www3/ (with custom filters)

Nested Folders

www1/ subfolder www2/ subfolder

API Endpoints

GET /api - Welcome message

About This Sample

This sample demonstrates serving static files in mORMot2, equivalent to DelphiMVCFramework's TMVCStaticFilesMiddleware.

The implementation uses a custom TRestHttpServer subclass that overrides the Request() method to intercept GET requests and serve files from the file system.
