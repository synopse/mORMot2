## Nature of This Work

This collection contains **ports**, not copies:

- **Original concepts**: Sample ideas, educational structure, and API patterns come from DMVC
- **New implementation**: All code is written from scratch using mORMot2 idioms
- **Purpose**: Demonstrate mORMot2 equivalents for developers familiar with DMVC

Each sample's README includes a link to the original DMVC sample for comparison.
