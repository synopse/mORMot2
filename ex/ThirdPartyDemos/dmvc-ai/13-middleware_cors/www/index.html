mORMot2 CORS Support Sample

⚠ Important: This page must be served from a different port (e.g., http://localhost:9090) than the API server (http://localhost:8080) to demonstrate CORS functionality correctly.
ℹ How to serve this page:
cd www
python -m http.server 9090
Then open http://localhost:9090

Change CORS configuration in the server:

Run the server with different modes:

  • MiddlewareCors all - Allow all origins (*) ✓
  • MiddlewareCors specific - Allow localhost:9090 + anotherserver.com ✓
  • MiddlewareCors restrict - Only anotherserver.com (blocks localhost:9090) ✗
